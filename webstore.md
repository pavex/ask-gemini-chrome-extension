# Gemini Quick Ask - Chrome Web Store Listing

## Short Description
Fast, private AI co-pilot. Ask Gemini about selected text or page content instantly using shortcuts or context menus.

## Detailed Description
Supercharge your browsing experience with **Gemini Quick Ask** – the high-performance, lightweight AI assistant that puts the power of Google's Gemini AI exactly where you need it. Designed for power users, researchers, and developers who value speed, privacy, and simplicity.

### 🚀 Instant Interaction
- **Smart Context:** Highlight any text and press **Ctrl+Shift+G** to get instant answers. No selection? No problem – the extension automatically understands the context of your entire page.
- **One-Click Prompts:** Save your most-used tasks (like "Summarize," "Refactor Code," or "Explain this") as custom buttons for immediate results.
- **Always Accessible:** Trigger via a keyboard shortcut, the right-click menu, or by clicking the extension icon.

### 🛡️ Privacy & Transparency (BYOK)
- **100% Free & Open:** The extension is completely free. No subscriptions, no hidden fees.
- **No Accounts Required:** No sign-ups, no logins, and zero tracking. We don't want your data.
- **Local Storage:** Your Gemini API Key and personal settings are stored **only in your browser's local storage**. 
- **Direct Connection:** Your requests go directly from your browser to Google's official API. Your data never touches our servers because we don't have any!

### ✨ Premium Features
- **Rich Markdown Support:** Responses are beautifully formatted with support for bold text, nested lists, and syntax-highlighted code blocks.
- **System Instructions:** Set a global baseline (e.g., "Always reply in Czech" or "Be very concise") that applies to all your queries.
- **UI Scaling:** Customize the chat window size to match your screen and comfort level.
- **"Thinking..." Feedback:** Real-time visual feedback while the AI generates your answer.

### ⚙️ How to Start:
1. Install **Gemini Quick Ask**.
2. Get your free API Key from [Google AI Studio](https://aistudio.google.com/app/apikey).
3. Paste the key into the extension settings.
4. Start browsing smarter!

*Note: This is a third-party extension and is not officially affiliated with Google. It requires your own Google Gemini API key to function.*
