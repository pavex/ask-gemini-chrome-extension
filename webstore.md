# Gemini Quick Ask - Store Listing

## Short Description
Quickly ask Gemini about selected text or page content using shortcuts, context menus, or a simple click.

## Detailed Description
Supercharge your browsing experience with Gemini Quick Ask! This lightweight, high-performance extension brings the power of Google's Gemini AI to any tab you're visiting. Whether you need to summarize a long article, explain complex code, or translate text, Gemini is just one click or keystroke away.

### 🚀 Key Features:
- **Instant Context:** Highlight any text and press Ctrl+Shift+G to start chatting. If nothing is selected, the extension automatically understands the context of the entire page.
- **One-Click Prompts:** Save your most-used prompts (like "Summarize this," "Explain like I'm five," or "Extract key points") as custom buttons for instant execution.
- **System Instructions:** Define a global "personality" for your AI (e.g., "Always reply in Spanish" or "Keep answers technical") that persists across all queries.
- **Rich Formatting:** Enjoy clean, readable responses with full Markdown support for lists, bold text, and code snippets.
- **Customizable UI:** Use the UI Scaling feature to make the chat window as large or small as you need.
- **Seamless Integration:** Access via keyboard shortcuts, the right-click context menu, or the extension bar.

### 🛡️ Privacy & Security:
- **BYOK (Bring Your Own Key):** This extension uses your personal Gemini API Key, giving you full control over your usage and costs.
- **Local Storage:** Your API key and settings are stored strictly in your browser's local storage. We never see or store your data.
- **Direct Connection:** All AI requests go directly from your browser to Google's official Gemini API.

### ⚙️ Quick Start:
1. Install the extension.
2. Get your free Gemini API key from Google AI Studio.
3. Paste the key into the extension settings.
4. Start asking!

Perfect for researchers, developers, students, and anyone who wants to browse smarter and faster.
